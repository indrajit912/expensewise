Secure Multi-User Auth

Every user's budget database is sandboxed. Hashed password keys, Talisman security headers, and brute-force limiters keep your account protected.

Backup & Restore Center

Securely backup your registry with portable JSON v2.0 files. Restores execute multi-point schema integrity checks and encrypt all data under active cryptographic keys.

Built for Developers

Manage your expenses directly from your terminal using our installable Command Line Interface (CLI) application. It connects securely to the ExpenseWise REST API.

  • Exposes a secure REST API (/api/v1/) with token auth.
  • Custom cli-client allows adding, deleting, listing expenses.
  • Supports terminal pagination and clean colored data rendering.
# Install the command line tool
$ pip install expensewise-cli
# Authenticate client wrapper
$ expensewise-cli login
Enter email: user@example.com
Enter password: *********
Success! Auth token stored locally.
# Check recent analytics
$ expensewise-cli analytics
