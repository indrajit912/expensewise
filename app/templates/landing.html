Standalone Terminal Client

Meet the ExpenseWise CLI

For developers, sysadmins, and keyboard-centric users, expensewise-cli is a standalone terminal client that handles registration, secure logins, transaction ledger listings, custom color categories, and budgeting comparisons—communicating exclusively via the versioned REST API.

  • Secure API Auth: Authenticate using API Tokens generated directly from settings.
  • Terminal Analytics & Charts: Generate beautiful, colorful Unicode bar charts mapping your spending category rollups.
  • Dynamic Budgeting: Compare budgets vs actual spending, retrieve averages, and update targets straight from your shell.
Get Started with CLI Get API Token
# Clone from GitHub
$ git clone https://github.com/indrajit912/expensewise.git
# Install CLI from source
$ cd expensewise/cli && pip install -e .
# Authenticate using your web API token
$ expensewise-cli token-login <your_api_token>
# Show current month's budget allowance tracker
$ expensewise-cli budget-show
# Generate colorful category spending chart
$ expensewise-cli chart
# List all expenses with pagination
$ expensewise-cli list
